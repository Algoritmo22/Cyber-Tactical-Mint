# Cyber Tactical Mint

Una personalización completa de Linux Mint 22.3 Cinnamon con estética táctica militar.

## Características

- Tema Green Tactical
- Kitty
- Conky
- btop
- Fastfetch
- Picom
- Plank
- Instalación automática

## Instalación

```bash
git clone https://github.com/Algoritmo22/Cyber-Tactical-Mint.git
cd Cyber-Tactical-Mint
./scripts/install.sh
```
